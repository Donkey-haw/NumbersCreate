# PDF to Numbers Converter (Hugging Face Edition)

This is a high-performance, multi-sheet PDF to Apple Numbers converter.
Enjoy **16GB RAM** for heavy textbook processing!

## Deployment
This repository is configured to run on Hugging Face Spaces using Docker.
It automatically pulls all dependencies and starts the FastAPI server on port 7860.
